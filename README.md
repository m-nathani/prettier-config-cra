# prettier-config-cra

> Create react app [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
$ npm i --save-dev prettier-config-cra
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "prettier-config-cra"
}
```

**Use any of the supported extensions to export a string, e.g. `.prettierrc`:**

```

```
