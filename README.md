# @umai/prettier-config

> UMAI [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
$ npm i --save-dev @umai/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@umai/prettier-config"
}
```
